# Big O Notations

Popular Big O Notations table.

| SYMBOL | NAME |
|--------|------|
| O(1) | Constant|
| O(logN)  | Logarithmic |
| O(N) | Linear |
| O(N * logN) |N log - star N|
| O(N^2) | Quadric |

Base on Image The Big notations time complexity order is like below:
```
O(1) < O(logN) < O(Sqrt(N)) < O(N) < O(N logN) < O(N^2) < O(2^N) < O(N!)

```
